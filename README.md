# 🌐 My Web — Personal Website & Portfolio

Selamat datang di **My Web**, sebuah website pribadi yang menampilkan proyek-proyek saya, keterampilan pengembangan web, dan pencapaian lainnya. Website ini dibangun dengan **Laravel**, **Vue.js**, dan menggunakan **Flux UI** untuk desain antarmuka yang modern dan responsif.

> 🔗 **Live Website:** [satyaadil.github.io/Myweb](https://satyaadil.github.io/Myweb/)

---

## ✨ Fitur Utama

- 🚀 **Frontend Modern & Responsif** menggunakan **Vue.js** dan **Flux UI**  
- ⚡ **Desain Minimalis** yang mudah dinavigasi dan mobile-friendly  
- 📂 **Portofolio Proyek** lengkap dengan deskripsi, teknologi yang digunakan, dan tampilan proyek  
- 🛠️ **Pengembangan Berbasis Laravel** dengan integrasi API untuk fitur dinamis  
- 🖥️ **Tampilan Profesional** yang sesuai untuk pengembang atau desainer

---

## 📦 Teknologi yang Digunakan

Website ini dibangun menggunakan teknologi dan alat terbaik untuk pengembangan web modern:

- **Backend:** Laravel 10
- **Frontend:** Vue.js, Flux UI
- **CSS Framework:** Tailwind CSS
- **Asset Bundling:** Vite
- **Database:** MySQL (opsional jika diperlukan untuk pengembangan fitur dinamis)
- **Deployment:** GitHub Pages / Hosting Kustom

---

## 📁 Struktur Proyek

Berikut adalah struktur direktori utama untuk proyek ini:

├── app/                 # Application logic (controllers, models, etc.)
├── bootstrap/           # Laravel bootstrap files
├── config/              # Application configuration files
├── database/            # Database migrations, factories, and seeds
├── public/              # Public files (index.php, asset files)
├── resources/           # Views and assets (CSS, JavaScript)
├── routes/              # Application routes
├── storage/             # File storage and logs
├── tests/               # Unit and feature tests
├── vite.config.js       # Vite configuration for asset bundling
├── package.json         # Front-end dependencies
├── composer.json        # Back-end dependencies
