# 🚀 Laravel Starter - Elegant & Modern Boilerplate

Selamat datang di **Laravel-Starter**, pondasi proyek Laravel yang disiapkan dengan arsitektur modern, performa optimal, dan desain frontend yang bisa kamu sesuaikan sesuai gaya!

> 🔗 Live Preview Website: [satyaadil.github.io/Myweb](https://satyaadil.github.io/Myweb/)

---

## ✨ Fitur Unggulan

- ✅ Laravel 10+ siap pakai
- ✅ Integrasi **Vite** untuk modern asset bundling
- ✅ Struktur proyek yang rapi dan scalable
- ✅ Siap dikembangkan dengan **Flux UI / shadcn-vue** (opsional)
- ✅ Cocok untuk pemula maupun pengembang berpengalaman
